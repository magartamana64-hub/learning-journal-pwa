Weekly Learning Journal

Each week, I reflect on my learning progress, challenges, and new skills gained in Mobile Application Development.

Week 1: Frontend Fundamentals (Temperature Converter PWA)

How did you approach mobile-first design?
I designed for mobile screens first with a simple layout and large buttons. Then I used media queries to adjust for bigger screens.

What was the most useful HTML or CSS concept you applied?
Flexbox helped align inputs and results neatly, keeping the layout responsive and clean.

What part of HTML or CSS did you find most challenging?
Media queries were tricky at first, especially maintaining layout consistency across different screen sizes.

Week 2 : Frontend Fundamentals

How did you approach mobile-first design?
I started designing for small screens first, ensuring content and layout were simple and functional. Then I used media queries to adjust for larger screens.

What was the most useful HTML or CSS concept you applied?
Flexbox was the most helpful tool for creating a clean and responsive navigation layout.

What part of HTML or CSS did you find most challenging?
I found media queries slightly tricky at first, especially balancing font sizes and spacing across devices.
